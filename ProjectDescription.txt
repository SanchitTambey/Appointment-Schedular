Developed a full-stack appointment scheduling application using ASP.NET Core, EF Core, SSMS, and React. The system allows users to securely log in, manage appointments (create, update, delete), and receive automated SMS reminders for upcoming appointments. Implemented user authentication and authorization for secure access, including password management features. Added search functionality to filter appointments and an auto-deletion feature for past appointments. Designed a responsive and user-friendly interface for seamless scheduling and management.
